## STEP 2: OPEN QGIS AND SET UP PROJECT

### 2.1 Start QGIS
1. Open QGIS Desktop
2. Create a New Project (Project ‚Üí New)

### 2.2 Set Project CRS
1. Go to: **Project ‚Üí Properties ‚Üí CRS**
2. Search for: **EPSG:4326**
3. Select: **WGS 84 (EPSG:4326)**
4. Click **OK**

**Why EPSG:4326?**
- Geographic coordinate system
- Uses latitude/longitude
- Standard for global maps
- Matches task requirements

---

## STEP 3: LOAD THE WORLD BOUNDARIES SHAPEFILE

### 3.1 Add Vector Layer
1. **Layer ‚Üí Add Layer ‚Üí Add Vector Layer**
2. Click **Browse** (...)
3. Navigate to your shapefile location
4. Select: **world-administrative-boundaries.shp**
5. Click **Add**
6. Click **Close**

### 3.2 Verify the Layer
- Layer should appear in the Layers Panel
- World map should be visible in the Map Canvas
- Check CRS matches EPSG:4326

### 3.3 Examine Attribute Table
1. Right-click on the layer ‚Üí **Open Attribute Table**
2. Look for country name field (likely: "name", "NAME", "country", "iso_a3", etc.)
3. Note the exact field name - you'll need this for joining!

**Common field names:**
- name / NAME
- country / COUNTRY
- iso_a3 (3-letter country code)
- admin / ADMIN

---

## STEP 4: IMPORT THE TOURIST DATA CSV

### 4.1 Add Delimited Text Layer
1. **Layer ‚Üí Add Layer ‚Üí Add Delimited Text Layer**
2. Click **Browse** and select: **SL_Tourists-2025.csv**
3. **File Format**: CSV
4. **Geometry Definition**: Select **No geometry (attribute only table)**
5. Click **Add**
6. Click **Close**

### 4.2 Verify CSV Data
1. Right-click on **SL_Tourists-2025** ‚Üí **Open Attribute Table**
2. Verify columns:
   - Country_Name
   - Total_Tourists_Arrivals
3. Check a few country names to ensure they match the shapefile

---

## STEP 5: JOIN THE DATA (CRITICAL STEP!)

### 5.1 Identify Join Fields
**Shapefile field:** The country name field you noted earlier
**CSV field:** Country_Name

### 5.2 Perform the Join
1. Right-click on **world-administrative-boundaries** layer
2. Select **Properties**
3. Go to **Joins** tab (left sidebar)
4. Click the **+** (Add join) button

### 5.3 Configure Join Settings
- **Join layer:** SL_Tourists-2025
- **Join field:** Country_Name
- **Target field:** [Your shapefile country field name]
- Check: ‚úì **Custom field name prefix** (leave empty or use "SL_")
- Click **OK**
- Click **OK** to close Properties

### 5.4 Verify Join Success
1. Open attribute table of world-administrative-boundaries
2. You should now see new columns:
   - SL_Tourists-2025_Country_Name
   - SL_Tourists-2025_Total_Tourists_Arrivals
3. Check that values appear for major countries

**Troubleshooting Join Issues:**
- Country names must match EXACTLY
- Check for spelling differences
- Check for extra spaces
- You may need to clean country names in QGIS or the CSV

## STEP 6: CLASSIFY THE MAP BY TOURIST ARRIVALS

### 6.1 Open Symbology
1. Right-click on **world-administrative-boundaries**
2. Select **Properties**
3. Go to **Symbology** tab

### 6.2 Set Graduated Symbology
1. At the top dropdown, change from **Single Symbol** to **Graduated**
2. **Value:** Select your joined field:
   - `SL_Tourists-2025_Total_Tourists_Arrivals`
   - Or similar based on your join prefix

### 6.3 Configure Classification
**Method Options:**

**Option A: Natural Breaks (Jenks)** - Recommended
- **Classes:** 5-7 classes
- **Mode:** Natural Breaks (Jenks)
- **Color ramp:** Choose a sequential scheme (e.g., Blues, YlOrRd)
- Click **Classify**

**Option B: Quantile**
- Good for equal distribution
- **Classes:** 5
- **Mode:** Quantile
- Click **Classify**

**Option C: Manual Classification**
Based on data distribution:
- Class 1: 1 - 100 (Very Low)
- Class 2: 101 - 1,000 (Low)
- Class 3: 1,001 - 10,000 (Medium)
- Class 4: 10,001 - 50,000 (High)
- Class 5: 50,001+ (Very High)

### 6.4 Customize Colors
1. **Color ramp:** Click dropdown
2. Choose: **Blues** or **YlOrRd** (Yellow-Orange-Red)
3. Check **Invert Color Ramp** if needed
4. Click **OK**

### 8.1 Layer Styling

## ALTERNATIVE: PYTHON/MATPLOTLIB APPROACH

### If you prefer Python for map creation:

```python
import pandas as pd
import matplotlib.pyplot as plt
import geopandas as gpd

# Read shapefile
world = gpd.read_file('world-administrative-boundaries.shp')

# Read CSV
tourists = pd.read_csv('SL_Tourists-2025.csv')

# Merge
world = world.merge(tourists, left_on='name', right_on='Country_Name', how='left')

# Plot
fig, ax = plt.subplots(1, 1, figsize=(20, 10))
world.plot(column='Total_Tourists_Arrivals', 
           ax=ax,
           legend=True,
           cmap='YlOrRd',
           edgecolor='black',
           linewidth=0.5,
           missing_kwds={'color': 'lightgrey'})

ax.set_title('Tourist Arrivals to Sri Lanka (Jan-Aug 2025)', fontsize=20)
ax.axis('off')
plt.savefig('task_b_map.png', dpi=300, bbox_inches='tight')
plt.show()
```

## COMMON ISSUES AND SOLUTIONS

### Issue 1: Countries Not Joining
**Solution:** 
- Check country name field in shapefile
- Compare names between CSV and shapefile
- Use ISO codes instead (iso_a3)

- Label only major countries
- Use smaller font
- Add buffer around labels
- Use rule-based labeling

### Issue 3: Colors Not Showing
**Solution:**
- Ensure join was successful
- Check field name in symbology
- Verify data is numeric
- Re-classify

### Issue 4: Scale Incorrect
**Solution:**
- Check CRS is EPSG:4326
- Reproject if needed
- Update scale bar units
